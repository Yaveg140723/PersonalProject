CSS Properties, Selector and their Specificity

Table of 10 Essential CSS Properties

Property Name Description Code Example
color Sets the text color color: blue;
Background-color Define the background color of an element Background-color: lightgrey;
font-size Speficies the text size font-size: 18px;
font-family Define the font type used font-family: Arial, Sans-serif;
text-align Sets text alignment (left, center, right) text-align: center;
margin add external spacing around an element margin: 10px;
padding Adds internal spacing inside the element padding: 5px;
border Adds a border around the element border: 2px solid black;
display Defines how an element is displayed (block, inline, flex, etc.) display: flex;
position Controls element placement (relative, absolute, fixed) position: absolute; top: 20px; left: 10px;

Table of Main CSS Selectors and Their Specificity

Selectors Name Description Code Example Specifity
*(Universal) Select all elements * { margin: 0; } 0, 0, 0, 0
element(type) Select all elements of the given type (p,div, etc.) p { color: red; } 0, 0, 0, 1
.class Select all elements with this class .title { font-size: 20px; } 0, 0, 1, 0
#id Select an element with this unique ID #main { background-color: yellow; } 0, 1, 0, 0
element.class Select a specific element with a class p.highlight { color: blue; } 0, 0, 1, 1
element#id Select a specific element with an ID div#container { padding: 10px; } 0, 1, 0, 1
attribute Select elements containing a given attribute [type="text"] { border: 1ps solid black; } 0, 0, 1, 0
pseudo-class Apply a style based on an element's state a:hover { color: green; } 0, 0, 1, 0
pseudo-element Target part of an element p::first-line { font-weight: bold; } 0, 0, 1, 0
combinators Select elements based on their relationship in the DOM div > p { margin: 10px; } Depends on structure
